Problemas:

Recuerda que la estructura compañera de las cadenas de caracteres es el ciclo while, y que la condición de terminación es el carácter especial '\0' (fin de cadena).

Los laboratorios Rick o Linno, expertos en biotecnología, han desarrollado una nueva técnica de conocer el sexo predominante en un ser humano. Esta técnica está basada en el número de cromosomas X y Y que tiene un ser humano. El sexo femenino está definido por un número mayor de cromosomas X que cromosomas Y, en el sexo masculino el número de cromosomas Y es mayor que el de los cromosomas X. Desafortunadamente no se puede concluir nada al respecto en caso de que los cromosomas X y Y estén balanceados, pues el sexo aún no está definido.

Desarrolla la función cromosomas_x que recibe una cadena de caracteres. La función regresa el número total de caracteres 'X' que hay en la cadena.

casos de prueba:
  • para YYYYYYYY = 0
  • para XXXXYYXY  = 5
  • para ABCDEFGI  = 0

Desarrolla la función cromosomas_y que recibe una cadena de caracteres. La función regresa el número total de caracteres 'Y' que hay en la cadena.


casos de prueba:
  • para YYYYYYYY = 8
  • para XXXXYYXY  = 3
  • para ABCDEFGI  =  0

Desarrolla el procedimiento determina_sexo que recibe una cadena de caracteres. El procedimiento utiliza las funciones cromosomas_x y cromosomas_y para desplegar en pantalla el sexo de un ser humano.

Entrada
 Salida

Desarrolla el procedimiento imprime_limpio, que recibe una cadena de caracteres como parámetro. El procedimiento despliega el contenido de la cadena sin espacios en blanco. Este procedimiento no modifica el arreglo que es pasado como parámetro, sólo ignora los espacios en blanco al momento de imprimir. Por ejemplo:

Entrada
 Salida
XXXX Y YX Y
 XXXXYYXY
X    YX    YY X   Y X
 XYXYYXYX

Desarrolla el procedimiento limpia_arreglo,  que recibe 2 cadenas de caracteres como parámetro. El procedimiento copia lo que esta en el arreglo de entrada en el de salida, pero elimina los espacios en blanco. Despues de mandar a llamar el procedimiento, imprime los 2 arreglos  para comparalos.  El encabezado para esta función puede ser:

void limpia_arreglo(char entrada[MAX], char salida[MAX])

 Arg_Salida

En el main() construye un menú que muestre las siguientes opciones:

MENú
1. Determinar el sexo
2. Imprimir sin espacios
3. Copiar arreglo sin espacios
 4. Salir
Opción?

De acuerdo a la opción seleccionada por el usuario utiliza la función apropiada de las implementadas anteriormente. Utiliza un ciclo do-while para desplegar el menú en pantalla hasta que el usuario seleccione la opción de Salir. Recuerda que la captura de datos y validación de valores debe ser realizada en el main( ), pues las funciones NO pueden realizar las tareas de validación o captura de datos.

Incluye el algoritmo como comentario antes de cada función.
