Operaciones con variables enteras

  1. Escribe un programa completo en C, que solicite 3 valores enteros al usuario, y que realice las operaciones aritméticas mostradas en la parte inferior. El resultado de cada operación debe ser desplegado en la pantalla (procura guardar el resultado de cada operación en un variable auxiliar aux antes de desplegar el resultado de cada operación).

Variables utilizadas : a, b, c y aux.

Operaciones :
  • A / B
  • A % B
  • A / B - C
  • A / ( B - C )

Analiza los resultados de cada operación para los valores de a = 10 y b = 5 y c = 2.

casos de prueba F(a,b,c) debe de imprimr en pantalla:
  • a) F(10, 5, 2) :
