Actividad: Autoestudio sobre el ciclo while.
Unidad: Estatutos de repetición.
Descripción
  • Realizarás el autoestudio que presenta el concepto de repetición (o ciclo), así como el primer ciclo que estaremos usando en C: while.
Modalidad
  • Individual.
Objetivos de la actividad
  • Conocer los conceptos de función y procedimiento, variable global y local. 
  • Diferenciar entre una función y un procedimiento, una variable global y una local.
  • Reconocer la importancia del uso de funciones y procedimiento para el diseño de un programa en C.
Instrucciones
Estudia del libro de texto (Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2010.) las siguientes secciones:
  • Sentencia while. (pp. 151)
Responde las siguientes preguntas:
    1. ¿Qué es un ciclo o repetición?
    2. ¿Qué es un centinela?
    3. ¿Para qué se usa una bandera?
    4. ¿Qué crees que suceda si al finalizar el encabezado del while se coloca ; (punto y coma)?
    5. ¿Como mejorarías el siguiente fragamento de código para que llegara a 10,000 elefantes? escríbe el código
