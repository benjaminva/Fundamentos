Unidad 1: Introducción.
Actividad: Instalación del ambiente de programación.
Descripción
  • Realizarás la instalación del ambiente de programación que estaremos usando a lo largo del semestre.
Modalidad
  • Individual.
Objetivos de la actividad
  • Instalación del ambiente de desarrollo.
Instrucciones
  • Baja Code Blocks que es el editor que usaremos este semestre (si el link no funciona busca codeblocks en internter con el compilador MinGw incluido) y sigue las instrucciones que aparecen en el asistente de instalación. Te recomendamos dejar las opciones por omisión que presente el asistente.
  • Una vez que termine el asistente, el ambiente de desarrollo se encontrará completamente instalado.
  • Para finalizar, cierra la aplicación.
Recursos
  • Programa de instalación que se encuentra en el área de "Material Adicional".
Especificaciones de entrega
  • No aplica.
